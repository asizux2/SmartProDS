# SKILL.md - MarBrain CLI

> **Skill**: marbrain-cli
> **Category**: research
> **Version**: 1.0.0

## Description

Unified CLI for MarBrain 360° Autonomous Research Platform. Provides command-line interfaces for all MarBrain agents including scraping, data pipeline, dashboard deployment, scheduling, deep dive execution, entity extraction, and learning loop optimization.

## Commands

### Web Scraping

```bash
marbrain scrape run <url> [--type TYPE]
marbrain scrape social <platform> <handles>
```

### Data Pipeline

```bash
marbrain pipeline clean [filepath]
marbrain pipeline validate <filepath>
```

### Dashboard Deployment

```bash
marbrain deploy dashboard <entity_name> [data_file]
marbrain deploy vercel
```

### Scheduling

```bash
marbrain schedule add <name> <type> <agent>
marbrain schedule list
marbrain schedule check
```

### Deep Dives

```bash
marbrain deepdive run <entity_name> [--dives N]
```

### Entity Extraction

```bash
marbrain extract entities <day_folder>
```

### Learning Loop

```bash
marbrain learn track <entity> [--success]
marbrain learn optimize
```

### Research

```bash
marbrain research <day>
marbrain status
```
