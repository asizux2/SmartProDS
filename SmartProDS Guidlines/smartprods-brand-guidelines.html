09 — Do's & Don'ts

Brand Usage Rules

These rules protect brand consistency across all touchpoints — web, email, presentations, and dashboards.

✓ DO
  • Use the exact hex values from this guide in all digital work
  • Use Electric Teal #00BFA6 as the primary brand color
  • Maintain minimum clear space around the logo at all times
  • Use JetBrains Mono for all numbers, statistics, and metric labels
  • Use the full wordmark "SmartProDS" — always one word, mixed case
  • Apply the dark navy background in Power BI dashboard builds
  • Use the teal-to-light-teal gradient for the icon mark only
  • Always pair the icon mark with the wordmark in formal contexts
  • Keep all type in approved font families
  • Ensure text contrast meets WCAG AA standards
✗ DON'T
  • Don't use NIQ Blurple #2D6DF6 — that's a competitor color
  • Don't rotate, distort, stretch, or recolor the logo
  • Don't use Inter, Roboto, or Arial — they're off-brand
  • Don't place the logo on busy backgrounds without a clear zone
  • Don't write "Smart Pro DS" or "smartprods" — it's always "SmartProDS"
  • Don't use gradient text for body copy — reserved for the icon only
  • Don't use purple or generic blue gradients anywhere
  • Don't use more than two accent colors in any single design
  • Don't use the amber color for primary CTAs — it's secondary only
  • Don't reduce the logo below minimum size requirements
