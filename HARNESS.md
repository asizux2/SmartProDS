# HARNESS.md - MarBrain CLI Harness

## What is this?

MarBrain CLI harness lets AI agents control the MarBrain 360° Autonomous Research Platform through standardized CLI commands.

## Why Generate?

This CLI enables any AI agent to:
1. Run research workflows automatically
2. Deploy dashboards to Vercel
3. Execute deep dive research cycles
4. Schedule continuous monitoring
5. Track learning and optimization

## Architecture

### Agents Covered

| Agent | CLI Command | Purpose |
|-------|-----------|---------|
| agent-scraper | `marbrain scrape` | Web scraping |
| agent-data-pipeline | `marbrain pipeline` | Data cleaning |
| agent-dashboard-deploy | `marbrain deploy` | Dashboard deployment |
| agent-scheduler | `marbrain schedule` | Cron scheduling |
| agent-deep-dive-runner | `marbrain deepdive` | Deep dive research |
| agent-entity-extractor | `marbrain extract` | Entity extraction |
| agent-learning-loop | `marbrain learn` | Optimization |

### Data Flow

```
AI Agent → marbrain CLI → MarBrain Core Agents → Research Data → Dashboard → Vercel
```

## Commands

### scrape
```bash
marbrain scrape run https://example.com --type ecommerce
marbrain scrape social twitter @talabat,@vodafone
```

### pipeline
```bash
marbrain pipeline clean raw_data.json
marbrain pipeline validate cleaned_data.json
```

### deploy
```bash
marbrain deploy dashboard "Talabat Egypt"
marbrain deploy vercel
```

### schedule
```bash
marbrain schedule add daily_scraper daily agent-scraper
marbrain schedule list
marbrain schedule check
```

### deepdive
```bash
marbrain deepdive run "Talabat Egypt" --dives 5
```

### extract
```bash
marbrain extract entities "DAY-001-Egypt-Ecommerce"
```

### learn
```bash
marbrain learn track "Talabat Egypt" --success
marbrain learn optimize
```

### research
```bash
marbrain research 153
marbrain status
```
